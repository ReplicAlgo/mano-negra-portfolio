Este Repository Mano-negra-portoflio muestra los resultados de aplicar el Indicador mano negra a 11 diferentes futuros financieros.  
Este projecto deja que el usuario combine diferentes instrumentos para identificar el portafolio que se ajuste mas a sus horarios y apetito de riesgo
Ideal para buscar instrumentos no correlacionados que mejoren el riesgo y tambien el retorno
